# 💡 Idea Vault

Idea Vault is a simple and organized place to capture, manage, and develop ideas. Whether you're collecting startup concepts, project plans, content ideas, or personal inspirations, Idea Vault helps you store everything in one searchable location.

## Features

- 📝 Create and save ideas
- 🏷️ Categorize ideas with tags
- ⭐ Mark favorite ideas
- 🔍 Search and filter ideas
- 📅 Track creation and update dates
- 📂 Organize ideas by categories
- 💾 Persistent storage
- 📱 Responsive and user-friendly interface

## Tech Stack

- Frontend: React / Next.js
- Backend: Node.js / Express
- Database: MongoDB / PostgreSQL
- Styling: Tailwind CSS
- Authentication: JWT / OAuth (optional)
